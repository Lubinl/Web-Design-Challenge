Summary: X vs. Latitude

The purposes of this project was to analyze how weather changes as you get closer to the equator. To acccomplish this analysis, we first pulled data from the OpenWeatherMap API to assemble a dataset of approximately 550 cities.

After Assembling the dataset, we used Matplotlib to plot various aspects of the weather vs. latitude.Factors we looked at included: temperature, cloudiness, wind speed,and humanity. This site provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correlations witnessed.
